# 使用 fscarmen 一键部署 sing-box 全家桶

在复杂的网络环境下，手动配置代理服务端往往费时费力。开源项目 `fscarmen` 提供了一款功能强大的 sing-box 一键部署脚本，支持多种现代代理协议（如 Vless、Reality、Hysteria2 等），极大地简化了部署流程。

## 部署流程

### 1. 执行一键安装脚本

在终端中输入并执行以下命令：

```bash
bash <(wget -qO- https://raw.githubusercontent.com/fscarmen/sing-box/main/sing-box.sh)
```

### 2. 交互式配置

执行脚本后，根据提示进行以下操作：

1. **选择语言**：脚本会请求选择显示语言（中文、英文等），根据个人喜好选择即可。
2. **选择安装模式**：在菜单中输入 `1` ，选择快速安装 sing-box 全家桶。
3. **获取节点信息**：安装完成后，控制台会输出各个客户端的订阅信息或节点链接。找到你所使用的代理软件对应的方框，完整复制其中的内容。

### 3. 客户端导入

打开你本地的代理客户端软件（如 v2rayN、Clash Verge 等），选择“从剪贴板导入”（或类似选项），即可完成节点的添加并开始使用。
