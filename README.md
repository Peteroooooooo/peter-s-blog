A high-tech, cyberpunk-themed personal portfolio showcasing a futuristic tactical UI, an interactive 3D core system, and immersive visual effects.

## üöÄ Features

- **Holographic 3D Core**: An interactive, dynamic 3D cube serving as the central navigation anchor and visual centerpiece.
- **Cyberpunk Aesthetics**: Custom CSS animations, CRT scanline overlays, glowing electric effects, and a pure "Terminal" vibe.
- **Artifacts Registry**: A 3D-perspective carousel showcasing past projects with a unique holographic projection effect.
- **System Logs**: Terminal-style technical logs and thought records featuring a seamless Mag-Lev scroll tracking indicator.
- **Interactive Personality**: Dynamic interactions tailored to different "Personality Modes" (Overclocked vs Idle, etc.).

## üíª Tech Stack

- **Core**: React 19 + TypeScript + Vite
- **Styling**: Tailwind CSS + Vanilla CSS Keyframes
- **3D & Rendering**: Three.js + React Three Fiber + Drei + Postprocessing
- **Animations**: GSAP + React Spring
- **Routing & State**: React Router DOM + Zustand

## üõ†Ô∏è Getting Started

### Prerequisites
- Node.js (v18 or newer recommended)

### Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/Peteroooooooo/peter-s-blog.git
   cd peter-s-blog
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to the local URL provided by Vite (usually `http://localhost:3000` or `http://localhost:5173`).
